Bienvenue sur le serveur de KickMyB

Incluant la documentation de l'API

API pour les comptes

Créer un compte :: POST /api/id/signup

{
  "username" : "MrPipo",
  "password" : "pipo"
}

Se connecter avec un compte existant :: POST /api/id/signin

{
  "username" : "MrPipo",
  "password" : "pipo"
}

Se déconnecter :: POST /api/id/signout

API pour les tâches

Ajouter une tâche au compte actuel (nécessite d'être connecté) :: POST /api/add

{
  "name" : "test",
  "deadline" : "2069-05-24T12:12:12"
}

Voir les tâches du compte actuel (nécessite d'être connecté) :: GET /api/home

voir le détail d'une tâche :: GET /api/detail/{id}

mettre à jour le % d'avancement :: GET /api/progress/{id}/{valeur}

id est le l'identifiant de la tâche et valeur est la nouvelle valeur d'avancement
